# KSArchive WIP
<p>Archiwum płyt CD i DVD z magazynów KomputerŚwiat od 2005-2015</p>
<p>BONUS! Polskie wersje Maluch Racera (MR1 jako plik ISO i MR2 jako instalator .EXE)</p>

<h2>Szybkie info:</h2>
<p>Programy na płytach <b>powinny</b> działać na Windows 98, Windows 2000, Windows XP, Windows Vista i Windows 7. (nie sprawdzone na Win98 i Win2000)</p>
<h3>Projekt jest zbiorem opublicznionych plików z internetu, szczegolnie z Internet Archive.</h3>
<b>Creditsy do riperów i crawlerów, bez nich repo by nie powstało❤️</b>

<h2>Instrukcje wypalania plików na fizyczną płytę CD/DVD </h2>
<p>1. Pobierz program <a href="https://download.imgburn.com/SetupImgBurn_2.5.8.0.exe">ImgBurn.</a></p>
<p>2. Zainstaluj i otwórz ImgBurn.</p>
<p>3. Włóż czystą płytę do napędu. <b>⚠️ UWAGA! Wypalenie płyty wszystko z niej usunie, tak jak formatowanie. ⚠️</b></p>
<p>4. Na ekranie startowym wybierz <b>Write image file to disc</b>.</p>
<p>5. W polu <b>Destination</b> wybierz swój napęd.</p>
<p>6. Kliknij w <b>ikonkę folderu z lupą w sekcji Source.</b></p>
<p>7. Wybierz swój plik .ISO</p>
<p>8. W sekcji <b>Settings</b> ustaw <b>Write speed</b> na 8x (jeśli masz nowszy napęd a program wywala błędy, nastaw na minimalną, znośną dla twojego napędu prędkość.)</p>
<p>9. Kliknij dużą ikonę wypalania, poczekaj na ukończenie i gotowe!</p>
