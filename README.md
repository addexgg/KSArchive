# KSArchive WIP

Archiwum płyt CD i DVD z magazynów KomputerŚwiat od 2005-2015  
**BONUS!** Polskie wersje Maluch Racera (MR1 jako plik ISO i MR2 jako instalator .EXE)

## Szybkie info:
Programy na płytach **powinny** działać na Windows 98, 2000, XP, Vista i 7 (nie sprawdzone na Win98/2000)  
### Projekt jest zbiorem opublicznionych plików z internetu, szczególnie z Internet Archive.  
**Creditsy do riperów i crawlerów, bez nich repo by nie powstało❤️**

## Instrukcje wypalania plików na fizyczną płytę CD/DVD
1. Pobierz program [ImgBurn](https://download.imgburn.com/SetupImgBurn_2.5.8.0.exe).  
2. Zainstaluj i otwórz ImgBurn.  
3. Włóż czystą płytę do napędu. **⚠️ UWAGA! Wypalenie płyty wszystko z niej usunie, tak jak formatowanie. ⚠️**  
4. Na ekranie startowym wybierz **Write image file to disc**.  
5. W polu **Destination** wybierz swój napęd.  
6. Kliknij w **ikonkę folderu z lupą w sekcji Source.**  
7. Wybierz swój plik .ISO  
8. W sekcji **Settings** ustaw **Write speed** na 8x (jeśli masz nowszy napęd a program wywala błędy, nastaw na minimalną, znośną dla twojego napędu prędkość)  
9. Kliknij dużą ikonę wypalania, poczekaj na ukończenie i gotowe!
